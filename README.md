# gestionIES

## 🛠️ Tecnologías principales

- **Frontend**: React + Vite + ShadCN (`src/`)  
- **Backend**: Node.js + Express (`backend/`)  
- **Base de datos**: PostgreSQL  
- **Autenticación**: LDAP
